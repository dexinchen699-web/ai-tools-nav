    前置条件：一个谷歌账号（Gmail 即可）；国内部分地区访问需要科学上网。

    🚀 Step by Step 详细操作

    1

    确认谷歌账号已登录

    打开浏览器，访问 gmail.com，确认右上角显示你的头像说明已登录。没有账号的话点「创建账号」，填邮箱和密码，约 2 分钟注册完成。

    2

    打开 Gemini 主界面

    访问 gemini.google.com，页面会自动用你的谷歌账号登录，进入对话界面——一个干净的输入框加左侧边栏。

    ⚠️

    国内部分地区需要科学上网才能访问。如果打不开，可以先用 DeepSeek（免费、无需梯子）作为替代方案。

    3

    切换到 Gemini 3.1 Pro

    在页面顶部找到模型名称（默认可能显示「Gemini」），点击下拉菜单，选择 Gemini 3.1 Pro。免费账号有一定的 3.1 Pro 使用额度，用完后会提示升级或自动回退基础版。

    💡

    看不到 3.1 Pro 选项？当前区域可能还未全量开放，先用最新的 Gemini 版本体验，核心功能差异不大。

    4

    发第一条消息，感受推理能力

    在底部输入框粘贴下面这条消息，按 Enter 发送：

    用三步解释量子纠缠是什么，每步用一个生活场景类比，最后一句话给出结论

    此时你应该看到：3–5 秒内出现结构清晰、有标题有分段的回答，类比生动，逻辑层次比普通 AI 回答明显更好。

    5A

    上传文件（PDF / 图片 / 视频）

    点击输入框左侧的 「+」图标（或回形针图标），选择本地文件上传。支持 PDF、Word、图片、视频等格式，上下文窗口高达 1M token（约 75 万字）。

    上传后发送：

    帮我用 5 条要点总结这份文档，每条不超过 2 句话

    此时你应该看到：Gemini 读取文档内容，输出结构化要点列表。遇到 50 页以上的 PDF，速度和准确率明显优于其他模型。

    5B

    开启 Deep Research（深度研究报告）

    在输入框下方找到 「🔬 深度研究」按钮，点击激活，然后输入研究主题，例如：

    2026 年全球储能行业现状与主要技术路线对比

    Gemini 会自动进行多轮联网搜索，整理成带引用链接的研究报告，通常需要 3–8 分钟。报告生成完成后可以一键导出到 Google Docs。

    💡

    Deep Research 适合写行业调研、竞品分析、论文综述，比手动搜索整理效率高 5–10 倍。生成的报告会列出所有引用来源，方便核实。

    5C

    （开发者）获取 Google AI Studio 免费 API

    打开 aistudio.google.com，用同一个 Google 账号登录，点击右上角 「Get API key」→「Create API key」，复制生成的密钥字符串。

    免费额度：每天 300 万 token，每分钟 5 次请求，用于个人项目和测试完全够用，不需要绑定任何付款方式。

    ℹ️

    只是日常使用的话，跳过这一步。这一步专门给想把 Gemini 接入自己工具、脚本或 Agent 的开发者用的。

    ❓ 常见问题

    国内用户需要梯子吗？
    访问 gemini.google.com 在国内大部分地区需要科学上网。Google AI Studio 部分地区可以直连。如果两者都无法访问，可以暂时用 DeepSeek 作为替代——完全免费、无需梯子、MIT 开源。
    免费版每天能用多少？
    免费版没有公开具体次数，轻度使用（10–20 次问答）通常不受影响。重度使用可能触达每日限额，届时会提示升级。AI Studio 免费 API 是每天 300 万 token、每分钟 5 次请求，轻量开发完全够用。
    和 ChatGPT 相比，哪个更好用？
    各有侧重：Gemini 3.1 Pro 在逻辑推理、长文档处理（1M 上下文）、Google 生态集成（Docs / Gmail / Drive）上更强；GPT-4.5 在对话自然度和幻觉控制上有优势，但 API 贵 60 倍。处理长文档或做研究选 Gemini，日常闲聊两者都行。
    上传的文件会被 Google 拿去训练模型吗？
    gemini.google.com 的对话内容默认可能被用于产品改进。进入「账号设置 → Gemini Apps Activity」关闭该选项即可避免。Google AI Studio 的 API 调用数据不会被用于训练，更适合处理敏感文件。
    Deep Research 生成的报告准确吗？
    整体准确率较高，但所有 AI 工具一样，数字和具体事实建议点开引用链接交叉验证。把它定位为「帮你做初步信息聚合」的工具，最终判断自己做，效果最好。
