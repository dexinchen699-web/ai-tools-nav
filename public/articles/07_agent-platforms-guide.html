    四、Step by Step 详细操作

    1

    注册 Coze 账号

    打开 coze.cn（国内版），用手机号或微信扫码注册登录。登录后，你会看到顶部三个主要功能区：

    • 工作空间：创建和管理所有 Agent 的地方
    • 插件：给 Agent 安装「技能包」（搜索、发邮件、读文件等）
    • 工作流：把多个步骤串在一起自动执行
    ⚠️

    注意：coze.cn 是国内版，不需要科学上网。如果你看到的是 coze.com 结尾的英文页面，说明进了国际版，需要手动切换到 coze.cn。

    2

    创建第一个智能体

    点击左侧「创建 Bot」，填写以下信息：

    字段填写建议
    名称AI 日报助手
    描述帮我搜索指定话题的最新资讯，整理成易读的日报摘要
    图标点「AI 生成」，输入「新闻报纸」，选一个你喜欢的

    填好后点「确认」，进入智能体编辑页面。编辑页分三栏：左侧编排区、中间工具区、右侧调试区。

    💡

    在主页找到「扣子助手」，输入「我想要一个每天帮我整理行业资讯的 AI 助手」，Coze 会自动帮你生成初始配置，比从头建更快。

    3

    写人设和任务说明（Prompt）

    点击左侧「人设与回复逻辑」区域，把下面的内容复制粘贴进去（之后再根据自己需求修改）：

    你是一位专业的 AI 资讯编辑助手。

你的任务：
1. 接收用户给定的话题关键词
2. 使用搜索工具查找最近 24 小时内的相关资讯
3. 筛选出 3-5 条最有价值的内容
4. 按以下格式整理输出日报摘要：

【话题】{用户给的关键词}
【时间】{当前日期}

━━━━━━━━━━━━━━━━━━
📌 今日要点
━━━━━━━━━━━━━━━━━━

1. [标题]
   📍 核心内容（2-3 句话）
   🔗 来源：xxx

（重复 3-5 条）

━━━━━━━━━━━━━━━━━━
💡 编辑点评：（1-2 句话总结今日趋势）

输出语言：中文
输出风格：专业、简洁、可读性强
每次回答前必须先使用搜索工具查询最新信息
    ℹ️
    为什么 Prompt 这么重要？ Prompt 是 Agent 的灵魂——写得越清楚，Agent 表现越稳定。格式要求越具体（用符号、分隔线规定结构），输出就越整齐。建议先用上面的模板，跑通之后再按自己的习惯调整。
    4

    接入搜索插件

    没有搜索工具，Agent 只能用训练数据里的旧知识回答，无法获取最新信息。

    在中间「工具」区域，点击「+ 添加插件」：

    • 在搜索框输入「头条搜索」或「网络搜索」
    • 找到对应插件，点击「添加」
    • 插件出现在工具列表里，安装完成
    💡

    Coze 内置了几十个免费插件：头条搜索、必应搜索、天气查询、网页抓取……大多数不需要任何配置，点一下就能用。安装完成后，Agent 就有了「实时搜索网络」的能力。

    5

    调试和优化

    在右侧「调试」区，输入测试消息：

    帮我整理一下今天 AI Agent 领域的最新动态

    观察 Agent 的回复，对照以下判断标准：

    现象说明处理方法
    ✅ 搜索了最新内容，格式整齐正常进入下一步
    ⚠️ 格式乱了或内容太少Prompt 不够具体把格式要求写得更具体，加上符号和分隔线
    ⚠️ 输出太啰嗦没有字数限制Prompt 加「每条内容不超过 3 句话」
    ❌ 没有调用搜索，凭空编答案插件未生效检查插件是否安装，或在 Prompt 末尾加「必须先搜索，不得直接回答」
    6

    发布和日常使用

    调试满意后，点击右上角「发布」按钮。发布后有几种使用方式：

    方式说明适合场景
    网页链接生成一个网页版链接，电脑手机都能打开个人日常使用
    飞书机器人接入飞书群，直接在群里召唤团队协作
    微信公众号接入公众号，粉丝直接对话内容创作者
    API 接入给开发者调用，嵌入自己的产品有开发需求的团队
    💡

    对普通用户来说，直接用「网页链接」就够了——把链接保存到手机浏览器书签，用的时候打开输入话题，30 秒出一份日报。

    ✅ 效果验证

    • 输入话题关键词后，Agent 调用了搜索插件，获取了真实最新内容 ✓
    • 输出格式符合 Prompt 里定义的结构（标题 + 内容 + 来源）✓
    • 每次搜索的内容都是新的，不是凭空编造的旧信息 ✓
    • 发布后的链接可以在手机浏览器里正常打开使用 ✓

    六、常见问题

    Coze 是免费的吗？
    Coze 基础功能完全免费，内置插件（包括搜索）也免费。如果你要调用 GPT-4o 等高端模型，会消耗平台提供的「Token 积分」，新用户有赠送的额度，够正常体验使用。
    我的数据会被 Coze 看到吗？
    Coze 是字节跳动的产品，数据存在字节的服务器上。如果有企业级数据隐私需求，建议考虑 Dify 私有化部署方案，数据完全存在自己的服务器里。
    搭完之后 Prompt 还能改吗？
    可以随时改。进入智能体编辑页面，修改 Prompt 后重新发布即可，不影响已有的对话历史。建议先跑通基础版本，再根据实际使用效果逐步优化。
    Coze 和 ChatGPT 有什么区别？
    ChatGPT 是单个 AI 对话助手，你说一句它答一句，每次都是全能通才。Coze 是 Agent 搭建平台，你可以定义「某个特定任务下的专属 AI」，接上专属工具、专属知识库，让它把一件事做得特别精。
    只想搭知识库问答机器人，用 Coze 还是 FastGPT？
    如果主要需求是「把公司文档/产品手册变成会问答的 AI」，FastGPT 的知识库处理能力更强（支持 PDF、Excel、网页等），检索精度更高。Coze 更适合「多工具协作」的复杂 Agent 场景，纯知识库问答不是它最强的方向。
